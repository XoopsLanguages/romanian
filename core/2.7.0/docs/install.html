Instalarea XOOPS

Prima instalare

Prefață:

XOOPS este o sursă deschisă Sistem de publicare Web orientat pe obiecte scris în PHP. Este un instrument ideal pentru dezvoltarea de site-uri web dinamice mici până la mari ale comunității, portaluri intra companie, portaluri corporative, bloguri și multe altele.

XOOPS este lansat în conformitate cu termenii Licență publică generală GNU (GPL) versiunea 2 sau mai mare și este liber de utilizat și modificat. Este gratuit să redistribuiți atâta timp cât respectați termenii de distribuție ai GPL.

Cerințe

  • Server WWW (Apache, NGINX, IIS, etc)
  • PHP 8.2 sau mai mare, 8.4+ recomandat
  • MySQL 5.7 .8 sau mai mare, recomandat MySQL 8.4.5+

Înainte de a instala

  1. Configurați corect serverul WWW, PHP și serverul de baze de date.
  2. Pregătiți o bază de date pentru site-ul dvs. XOOPS.
  3. Pregătiți contul de utilizator și acordați utilizatorului acces la baza de date. butonul radio
  4. Faceți directoarele și fișierele inscriptibile: %s
  5. Pentru considerente de securitate, vă recomandăm să mutați cele două directoare de mai jos din rădăcină document și schimbați numele folderelor: %s
  6. Creați (dacă nu este deja prezent) și faceți directoarele inscriptibile: %s
  7. Activați cookie-urile și JavaScript ale browserului dvs.

Se instalează local

Dacă rulați un mediu local pentru dezvoltare sau testare, asigurați-vă că aveți cel anterior cerințe îndeplinite. După ce ați făcut acest lucru, copiați conținutul fișierului htdocs (din XOOPS fișier de distribuție) către calea documentului rădăcină a mediului dvs. web. Odată ce fișierele sunt copiate acolo, puteți începe instalarea îndreptând browserul către „http://site-ul tău.com". Aceasta va începe procesul de instalare.

Instalare pe o platformă găzduită

Dacă rulați într-un mediu găzduit, despachetați fișierele XOOPS local sau pe server dacă aveți acces shell (adică SSH). După ce ați terminat, asigurați-vă că mutați sau copiați toate fișierele XOOPS din htdocs în directorul web rădăcină (furnizorul dvs. furnizează de obicei această locație cu indicații). Odată ce fișierele sunt copiate acolo, puteți începe instalarea indicând browserul dvs "http://site-ul tău.com". Aceasta va începe procesul de instalare.
