Ajutor: Profil

-----Conținut de ajutor -----------

DESCRIERE

Modulul Profil este pentru gestionarea câmpurilor personalizate de profil de utilizator.

INSTALARE/DEZINSTALARE

Nu sunt necesare măsuri speciale, urmați procesul standard de instalare – extrageți folderul /profile în directorul ../modules. Instalați modulul prin Admin -> Modul de sistem -> Module.

Instrucțiuni detaliate despre instalarea modulelor sunt disponibile în Manual de operațiuni XOOPS

INSTRUCȚIUNI DE UTILIZARE

Pentru a configura acest modul, trebuie să:

  • Configurați preferințele pentru modul (vezi „Preferințe”) și opțional blocul Profil dacă intenționați să îl utilizați (vezi „Blocuri”).
  • Editați categoriile existente sau adăugați altele noi.
  • Editați câmpurile existente sau adăugați altele noi. Aici puteți specifica ce câmpuri vor fi vizibile în ce categorie și dacă vor fi vizibile în timpul înregistrării utilizatorului.
  • Definiți ordinea pașilor de înregistrare.
  • Și, în sfârșit, puteți seta permisiuni pentru câmpurile individuale - care sunt editabile, care pot fi căutate.

Măsuri anti-spam

Pentru a minimiza înregistrările spam, procedați în felul următor:

  • accesați modulul Protector din Admin, mergeți la Preferințe, apoi în partea de jos, la această opțiune: „Opriți spamul pe forum” Verifică datele POST cu spammerii înregistrați în baza de date www.stopforumspam.com.
    Setați-l la „Ban IP-ul (fără limită)”

  • în /class/captcha/config.php, asigurați-vă că modul este setat ca „text”:

    returnează $config = matrice(
    'disabled' => false, // Dezactivează CAPTCHA
    'mode' => 'text', // modul implicit, puteți alege 'text', 'image', 'recaptcha' (necesită cheia API)
    'name' => 'xoopscaptcha', // nume captcha
    'skipmember' => true, // Omite verificarea CAPTCHA pentru membri
    'maxattempts' => 10, // Încercări maxime pentru fiecare sesiune
    );

  • În modulul Profil, accesați Admin și în pasul de bază situat la:
    /modules/profile/admin/step.php?id=1
    setați „Salvare după oprire” la „Nu”


  • În Preferințe profil, setați opțiunea „Utilizați Captcha după al doilea pas de înregistrare” la „Da” (este valoarea implicită)
