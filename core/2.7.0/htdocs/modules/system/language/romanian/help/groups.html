Ajutor: Grupuri

Descriere

XOOPS încorporează un sistem de înregistrare a membrilor - poți solicita opțional oamenilor să se înscrie pe site-ul tău pentru a avea acces la anumite zone sau funcții. De exemplu, majoritatea site-urilor solicită oamenilor să se alăture înainte de a li se permite să contribuie cu știri sau să posteze mesaje pe forumuri.
Drepturile de acces și de administrare la un site web XOOPS sunt setate printr-un sistem de permisiuni flexibil bazat pe „grupuri de utilizatori”. Grupurile implicite includ utilizatori anonimi, membri înregistrați și webmasteri, dar desigur că puteți defini câte grupuri aveți nevoie pentru a vă gestiona site-ul. Fiecare grup poate să primească un set unic de drepturi de acces care reglementează:

- Ce blocuri pot fi văzute
- Ce module pot fi accesate
- Ce module pot fi administrate
- Ce aspecte ale administrării sistemului pot fi modificate

Puteți schimba numele acestor grupuri implicite, dar nu le puteți șterge. Alte grupuri de utilizatori pot fi, de asemenea, setate și definite, așa cum se discută mai târziu.

Grupurile personalizate pot fi create și prin intermediul paginii de administrare a grupurilor.
